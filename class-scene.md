# 科目間のつながりと授業の雰囲気

## 科目間のつながり

- 1年前期の社会情報科学概論では，1年後期以降の科目で学ぶ内容の概要をつかみます．
- 1年後期の確率・統計は，2年生以降のデータ分析演習や機械学習などで必要な統計学の基礎を学びます．
